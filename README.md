## C'est quoi ce projet ?

Dans ce projet, nous avons appris à utiliser le **terminal Linux** et à écrire des **scripts Bash**. Le Bash est un langage qui permet d'automatiser des tâches sur un système Linux, c'est-à-dire d'écrire des commandes dans un fichier pour les exécuter automatiquement sans avoir à tout retaper à la main.

Le projet est divisé en deux parties : d'abord on a appris les commandes Linux de base, ensuite on a créé des scripts plus complexes.

---

## Les commandes Linux

On a commencé par découvrir les commandes essentielles pour naviguer et travailler dans un système Linux :

- **Lister les fichiers** : afficher le contenu d'un dossier avec la commande `ls`
- **Lire le contenu d'un fichier** : ouvrir et afficher un fichier directement dans le terminal avec `cat` ou `less`
- **Installer et mettre à jour les paquets** : gérer les logiciels installés sur le système avec `apt`
- **Utilisateur et permissions** : créer des utilisateurs, gérer qui a le droit de lire, écrire ou exécuter un fichier
- **Créer des raccourcis** : créer des alias pour aller plus vite dans le terminal

---

## Les scripts Bash

Ensuite on a écrit nos propres scripts, c'est-à-dire des petits programmes en Bash :

- **Mettre à jour le système** : un script qui lance automatiquement la mise à jour de tous les paquets
- **Copier le contenu d'un fichier** : un script qui copie des fichiers d'un endroit à un autre
- **Créer une calculatrice** : un script interactif qui demande deux nombres et effectue des opérations mathématiques
- **Script périodique** : un script qui s'exécute automatiquement à intervalles réguliers grâce à `cron`
- **Vérifier un format** : un script qui vérifie si une saisie correspond bien à un format attendu (par exemple une adresse email ou une date)
